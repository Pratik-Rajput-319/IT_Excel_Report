# IT_Excel_Report
Automated Financial Dashboard in Excel 
Built an automated dashboard to analyze quarterly financial performance of top Indian IT companies using Excel, Power Query, Pivot Tables, and interactive charts.
Automated data extraction and transformation workflow from multiple quarterly source files using Power Query.
Reduced manual reporting effort through one-click dashboard refresh automation.
Tools used were Excel, Power Query, Pivot Tables, Data Cleaning,Dashboarding.

The Main Steps to build this was "Source Files → Power Query → Pivot Tables → Charts → Automated Dashboard".
